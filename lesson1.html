Versionsverwaltung mit Git(Hub)

Git ist eine Versionsverwaltungssoftware

  • Open Source
  • Verteilte Versionskontrolle
  • Hohe Effizenz bei geringem Fingerabdruck

Git ist der Quasi-Standard. Viele Open-Source-Software werden auf GitHub entwickelt.

Aufgaben

  1. Öffne GitHub und registriere dich oder melde dich an.
  2. Arbeite den Hello-World-Guide durch.
  3. Öffne das Repository marcelreschke/cod203
  4. Forke das Repository.
  5. Öffne den Terminal und erstelle mit dem Befehl mkdir sites einen Arbeitsordner
  6. Wechsle mit dem Befehl cd sites in den Ordner.
  7. Clone dein geforktes Repository mit dem Befehl git clone https://github.com/dein_benutzername/cod203.git auf in das Verzeichnis cod203.
  8. Wechsle mit dem Befehl cd cod203 in den Ordner.
  9. Öffne in VSC den Ordner.
  10. Schaue dir die Datei index.html an.
