Dokumente auszeichnen mit HTML

HTML ist eine Auszeichnugssprache. Im Gegensatz zu einer Programmiersprache, können mit HTML keine Algorithmen erstellt und ausgeführt werden.

HTML wird in der Web-Entwicklung zur Auszeichnung der Informationen im Browser verwandt.

HTML-Tags

HTML nutzt Tags zur Auszeichnung der Inhalten. Sie bestehen aus in Spitzklammern umschlossenen Element-Namen

HTML-Tags kommen meistens in Paaren wie z. B. <p> und </p>.

HTML-Tags können Attribute aufweisen. Attribute bestehen aus einem Schlüssel/Wert-Paar wie z. B. <a title="Link-Titel für Tooltip">
